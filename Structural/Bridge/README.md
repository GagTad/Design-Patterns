# Bridge Design Pattern - C++ Example


## File Descriptions

*   **`IDevice.h`**: Abstract interface for devices.
*   **`TV.h / .cpp`**: Implementation of the TV device.
*   **`Radio.h / .cpp`**: Implementation of the Radio device.
*   **`RemoteControl.h / .cpp`**: Base abstraction class containing the bridge reference.
*   **`AdvancedRemote.h / .cpp`**: Refined abstraction with extra features.
*   **`main.cpp`**: Entry point demonstrating the interaction between different remotes and devices.
